Tic-Tac-Toe Game

Project Description

This project is a terminal-based 2-player Tic-Tac-Toe game written in Python. Players take turns entering the row and column where they want to place their mark, X or O. The program displays the board after each turn, validates user input, prevents invalid moves, detects winning combinations across rows, columns, and diagonals, and announces a draw when the board is full with no winner.

Instructions to Run the Program

Open a terminal.

Navigate to the project folder:
cd "/Users/charviiruganti/Desktop/cmpsc132/Project1"

Run the program with Python 3:
python3 tic_tac_toe.py

Enter the row and column separately, using numbers from 0 to 2.

Features

- 3x3 Tic-Tac-Toe board displayed in the terminal
- Two-player gameplay using X and O
- Input validation for invalid and out-of-range values
- Protection against choosing an occupied cell
- Automatic win detection for rows, columns, and diagonals
- Draw detection when the board is full
- Replay option after each game
